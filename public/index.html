Task App - Rest Api

You have to provide token in header for Authorized routes with Bearer keyword. Example : Bearer sometokenhere

Here is the list of all routes:

  • Unauthorized:
  • /users - POST
  • /users/login - POST
  • /users/:id/avatar - GET
  • Authorized:
  • /users/logout - POST
  • /users/logoutAll - POST
  • /users/me - GET , DELETE , PATCH
  • /users/me/avatar - POST , DELETE
  • /tasks - GET , POST
  • /tasks/:id - GET , PATCH , DELETE
